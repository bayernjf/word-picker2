# WordCatcher 手动测试清单

## 核心交互

1. 打开任意英文网页。
2. 按住 `Ctrl`，确认光标变为笔形。
3. 将鼠标停留在英文单词上约 300ms。
4. 确认弹出翻译弹窗，并显示单词、音标、释义。
5. 松开 `Ctrl`，确认弹窗关闭、光标恢复默认。

## 保存与回顾

1. 在弹窗中点击“添加到单词本”。
2. 确认按钮提示“已添加到单词本”或“单词已存在”。
3. 点击扩展图标，确认 Popup 中可看到新单词。
4. 使用搜索框按单词或释义筛选。
5. 删除某条记录，确认列表即时刷新。

## 导出

1. 在 Popup 中点击“导出 JSON”。
2. 确认浏览器下载 `wordcatcher-words.json`。
3. 再点击“导出 CSV”。
4. 确认浏览器下载 `wordcatcher-words.csv`。

## 设置

1. 打开扩展设置页。
2. 修改查词键、悬停延迟和自动朗读选项。
3. 点击保存，确认页面提示“设置已保存”。
4. 返回网页重新测试，确认新设置已生效。

## 翻译接口

1. 不配置任何密钥，确认弹窗可直接返回免费翻译结果。
2. 临时断网或接口异常时，确认弹窗显示兜底释义或降级提示。
3. 重复查询同一单词，确认响应更快，说明缓存已生效。
